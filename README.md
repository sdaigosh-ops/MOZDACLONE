# mozda-mindmap

**目的：** 週次会話データからMozdaの思考パターンを蓄積し、将来のローカルAIクローンの素材とする。

## 運用フロー

```
毎週（日曜推奨）
Claude + ChatGPT の週間会話を振り返り
    ↓
各AIに週次レポート生成を依頼
    ↓
このリポジトリにpush（weekly/YYYY-WXX.md）
    ↓
patternsとreactionsを随時更新
```

## ディレクトリ構造

```
mozda-mindmap/
├── weekly/          # 週次スナップショット
├── patterns/        # 蓄積された思考パターン（長期更新）
├── reactions/       # 反応データ集（好き嫌い・理由）
├── PROMPT.md        # AIへの週次レポート依頼プロンプト
└── SUMMARY.md       # 現時点の全体像（随時更新）
```

## クローン化ロードマップ

| フェーズ | 期間 | 内容 |
|---------|------|------|
| データ収集 | 〜6ヶ月 | 週次レポート蓄積 |
| パターン抽出 | 3ヶ月目〜 | patterns/ の充実 |
| fine-tuning準備 | 5ヶ月目〜 | ローカルLLM選定・データ整形 |
| クローンv0.1 | 6ヶ月後 | 試験運用開始 |

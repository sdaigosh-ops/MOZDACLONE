# 週次レポート依頼プロンプト

## Claudeへの依頼文（毎週コピペ）

```
今週の会話を振り返って、以下の形式でweekly reportを生成してください。
Markdownで出力。ファイル名は YYYY-WXX.md 形式。

抽出項目：
1. 今週の主な判断・決定とその根拠
2. 深く反応したアイデア・概念（なぜ響いたか）
3. 却下・保留したものとその理由
4. 新しく現れた視点・変化
5. 繰り返し使った比喩・言葉の癖
6. 今週の「核心的な一文」（最も本質を表す発言）
```

## ChatGPTへの依頼文

```
今週の会話から私の思考パターンレポートを作成してください。
以下を含めてください：
- 意思決定の傾向
- 何に時間と思考を使ったか
- 避けたこと・後回しにしたこと
- 感情的な反応が強かった話題
- Claudeのレポートと比較したときの差異（あれば）
```

## 統合メモ

- ClaudeとChatGPTの視点が**食い違う部分**が特に重要
- 食い違いは「どちらが本質を捉えているか」ではなく両方保存する
